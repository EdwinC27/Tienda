Los lenguajes de programación se conocen como programas que sirve para construir otros programas informáticos. Para entendernos, se trata de un lenguaje formal que ha sido creado para organizar algoritmos y procesos lógicos que se ejecutarán por un ordenador o sistema informático. Un lenguaje de programación es un lenguaje formal que le proporciona a una persona, en este caso el programador.
